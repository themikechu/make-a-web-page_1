Languages and other Stuff

Three Languages

HTML, Hyper Text Markup Language, is the standard markup language.

CSS, Cascading Style Sheets, describes the look and feel.

Java, which we haven't learned too much about yet.

DOM

DOM, Document Object Model, we learned, is a tree like structure that code can be organized into.

Boxes

Websites are designed by boxes. Boxes are arranged to organize and display the website.

Ways to Have CSS
  1. Write CSS into the "head of your HTML"
  2. Link your HTML to a separate CSS file (place link in head)
  3. Write your style inline. This leads to alot of repeated code.
Repetition

If you can create an organized structure, with code grouped into classes, then you can affect different aspects of code with changes in the CSS. This helps to minimize the amount of repetition and amount of code.
